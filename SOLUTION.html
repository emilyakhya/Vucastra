2.2.1 VUCA Components

Komponen VUCA didefinisikan secara matematis sebagai:

Volatility Index:

$$V = \frac{\sigma}{\mu} \times 100\%$$

Uncertainty Index:

$$U = -\sum_{i=1}^{n} p_i \log_2(p_i)$$

Complexity Index:

$$C = \frac{\log_2(N)}{L}$$

Ambiguity Index:

$$A = \frac{\sum_{i=1}^{n} u_i \log(u_i)}{\log(n)}$$

VUCA Composite Index:

$$\text{VUCA}_{\text{composite}} = \sum_{i=1}^{4} w_i \cdot \text{VUCA}_i$$

Dimana $w_i$ adalah bobot untuk setiap komponen dengan $\sum_{i=1}^{4} w_i = 1$

2.2.2 MPC Components

Economic Capability:

$$E = \frac{\sum_{i=1}^{k} \alpha_i \cdot E_i}{\sum_{i=1}^{k} \alpha_i}$$

Political Capability:

$$P = \frac{\sum_{i=1}^{k} \beta_i \cdot P_i}{\sum_{i=1}^{k} \beta_i}$$

Security Capability:

$$S = \frac{\sum_{i=1}^{k} \gamma_i \cdot S_i}{\sum_{i=1}^{k} \gamma_i}$$

Diplomatic Capability:

$$D = \frac{\sum_{i=1}^{k} \delta_i \cdot D_i}{\sum_{i=1}^{k} \delta_i}$$

MPC Composite Score:

$$\text{MPC} = \alpha \cdot E + \beta \cdot P + \gamma \cdot S + \delta \cdot D$$

Dengan bobot default: $\mathbf{w}_{\text{MPC}} = [0.30, 0.25, 0.25, 0.20]^T$

3. MATHEMATICAL FRAMEWORK DAN MODEL

3.1 VUCA Index Mathematical Framework

3.1.1 Volatility Index (V)

Volatility Index mengukur perubahan cepat dalam geopolitik berdasarkan koefisien variasi:

$$V = \frac{\sigma}{\mu} \times 100\%$$

Dimana:

  • $\sigma$ = Standard deviation dari time series
  • $\mu$ = Mean dari time series

Implementation:

def calculate_volatility_index(time_series):
    """
    Calculate Volatility Index based on coefficient of variation
    """
    mean = np.mean(time_series)
    std = np.std(time_series)
    volatility_index = (std / mean) * 100 if mean != 0 else 0
    return volatility_index

3.1.2 Uncertainty Index (U)

Uncertainty Index menggunakan Shannon entropy untuk mengukur ketidakpastian:

$$U = -\sum_{i=1}^{n} p_i \log_2(p_i)$$

Dimana:

  • $p_i$ = Probability dari event $i$ yang terjadi
  • $\log_2$ = Binary logarithm
  • $n$ = Jumlah possible events

Implementation:

def calculate_uncertainty_index(probabilities):
    """
    Calculate Uncertainty Index using Shannon entropy
    """
    uncertainty = 0
    for p in probabilities:
        if p > 0:
            uncertainty -= p * np.log2(p)
    return uncertainty

3.1.3 Complexity Index (C)

Complexity Index menggunakan information theory untuk mengukur kompleksitas sistem:

$$C = \frac{\log_2(N)}{L}$$

Dimana:

  • $N$ = Jumlah distinct elements atau states
  • $L$ = Average length dari patterns atau sequences

Implementation:

def calculate_complexity_index(elements, pattern_lengths):
    """
    Calculate Complexity Index using information theory
    """
    N = len(set(elements))
    L = np.mean(pattern_lengths) if pattern_lengths else 1
    complexity_index = np.log2(N) / L if L > 0 else 0
    return complexity_index

3.1.4 Ambiguity Index (A)

Ambiguity Index mengukur ambiguitas interpretasi dan makna:

$$A = \frac{\sum_{i=1}^{n} u_i \log(u_i)}{\log(n)}$$

Dimana:

  • $u_i$ = Uncertainty level untuk factor $i$
  • $n$ = Jumlah factors yang dipertimbangkan

Implementation:

def calculate_ambiguity_index(uncertainty_levels):
    """
    Calculate Ambiguity Index for interpretative clarity
    """
    n = len(uncertainty_levels)
    if n <= 1:
        return 0
    
    ambiguity = 0
    for ui in uncertainty_levels:
        if ui > 0:
            ambiguity += ui * np.log(ui)
    
    ambiguity_index = ambiguity / np.log(n)
    return ambiguity_index

3.1.5 VUCA Composite Index

VUCA Composite Index menggabungkan semua komponen dengan bobot yang dapat disesuaikan:

$$\text{VUCA}_{\text{composite}} = \sum_{i=1}^{4} w_i \cdot \text{VUCA}_i$$

Dimana:

  • $w_i$ = Bobot untuk komponen $i$ dengan $\sum_{i=1}^{4} w_i = 1$
  • $\text{VUCA}_i$ = Normalized value dari komponen $i$

Default Weights:

$$\mathbf{w} = [0.25, 0.25, 0.25, 0.25]^T$$

3.2 Middle Power Capability (MPC) Mathematical Framework

3.2.1 Economic Capability Score (E)

Economic Capability Score mengukur kapabilitas ekonomi berdasarkan multiple indicators:

$$E = \frac{\sum_{i=1}^{k} \alpha_i \cdot E_i}{\sum_{i=1}^{k} \alpha_i}$$

Dimana:

  • $\alpha_i$ = Bobot untuk economic indicator $i$
  • $E_i$ = Normalized value dari economic indicator $i$

Economic Indicators:

  • GDP per capita
  • Trade volume
  • Foreign Direct Investment (FDI)
  • Economic growth rate

3.2.2 MPC Composite Score

MPC Composite Score menggabungkan semua capability scores dengan bobot yang dapat disesuaikan:

$$\text{MPC} = \alpha \cdot E + \beta \cdot P + \gamma \cdot S + \delta \cdot D$$

Default Weights:

$$\mathbf{w}_{\text{MPC}} = [0.30, 0.25, 0.25, 0.20]^T$$

5.2 Random Forest for VUCA Classification

Model: Random Forest Classifier

Mathematical Foundation:

Random Forest menggunakan ensemble dari multiple decision trees:

$$\hat{y} = \text{mode}\{h_1(x), h_2(x), \ldots, h_T(x)\}$$

Ensemble Probability:

$$P(y = c|x) = \frac{1}{T}\sum_{t=1}^{T} \mathbb{I}[h_t(x) = c]$$

Dimana:

  • $h_t(x)$ = Prediction dari tree ke-$t$
  • $T$ = Total jumlah trees
  • $\mathbb{I}[\cdot]$ = Indicator function

Implementation:

def train_vuca_classifier(X_train, y_train, hyperparameters=None):
    from sklearn.ensemble import RandomForestClassifier
    
    if hyperparameters is None:
        hyperparameters = {
            'n_estimators': 100,
            'max_depth': None,
            'min_samples_split': 2,
            'min_samples_leaf': 1,
            'random_state': 42
        }
    
    rf_classifier = RandomForestClassifier(**hyperparameters)
    rf_classifier.fit(X_train, y_train)
    
    return rf_classifier

5.3 Neural Network for MPC Prediction

Model: Multi-layer Perceptron (MLP)

Mathematical Foundation:

Neural network dengan activation function $\sigma$ dan multiple hidden layers:

$$f(x) = W_L \cdot \sigma(W_{L-1} \cdot \sigma(\ldots \sigma(W_1 \cdot x + b_1) \ldots) + b_{L-1}) + b_L$$

Dimana:

  • $W_i$ = Weight matrix untuk layer ke-$i$
  • $b_i$ = Bias vector untuk layer ke-$i$
  • $\sigma$ = Activation function (ReLU)
  • $L$ = Total jumlah layers

Implementation:

def build_mpc_neural_network(input_dim, hidden_layers=[64, 32, 16]):
    from tensorflow.keras.models import Sequential
    from tensorflow.keras.layers import Dense, Dropout
    
    model = Sequential()
    
    # Input layer
    model.add(Dense(hidden_layers[0], input_dim=input_dim, activation='relu'))
    model.add(Dropout(0.2))
    
    # Hidden layers
    for units in hidden_layers[1:]:
        model.add(Dense(units, activation='relu'))
        model.add(Dropout(0.2))
    
    # Output layer
    model.add(Dense(1, activation='linear'))
    
    model.compile(optimizer='adam', loss='mse', metrics=['mae'])
    return model

5.4 Time Series Models

5.4.1 ARIMA Model

Formula: ARIMA(p,d,q)

Mathematical Formulation:

$$\phi(B)(1-B)^d X_t = \theta(B) \epsilon_t$$

Dimana:

  • $\phi(B)$ = Autoregressive polynomial dengan order $p$
  • $(1-B)^d$ = Differencing operator dengan order $d$
  • $\theta(B)$ = Moving average polynomial dengan order $q$
  • $\epsilon_t$ = White noise error term

Implementation:

def fit_arima_model(time_series, order=(1,1,1)):
    from statsmodels.tsa.arima.model import ARIMA
    
    model = ARIMA(time_series, order=order)
    fitted_model = model.fit()
    
    return fitted_model

5.4.2 VAR Model

Formula: VAR(p)

Mathematical Formulation:

$$\mathbf{X}_t = \mathbf{c} + \sum_{i=1}^{p} \mathbf{A}_i \mathbf{X}_{t-i} + \boldsymbol{\epsilon}_t$$

Dimana:

  • $\mathbf{X}_t$ = Vector dari time series pada waktu $t$
  • $\mathbf{c}$ = Constant vector
  • $\mathbf{A}_i$ = Coefficient matrix untuk lag $i$
  • $\boldsymbol{\epsilon}_t$ = Error vector

Implementation:

def fit_var_model(time_series, maxlags=5):
    from statsmodels.tsa.vector_ar.var_model import VAR
    
    model = VAR(time_series)
    results = model.select_order(maxlags)
    fitted_model = model.fit(maxlags=results.aic)
    
    return fitted_model

6. OPTIMIZATION ENGINE DAN RESOURCE ALLOCATION

6.1 Linear Programming Model

Mathematical Formulation:

Optimasi alokasi sumber daya menggunakan linear programming:

$$\max_{\mathbf{x}} \quad f(\mathbf{x}) = \mathbf{w}^T \mathbf{x}$$

Subject to:

$$\mathbf{A} \mathbf{x} \leq \mathbf{b}$$ $$\mathbf{x} \geq \mathbf{0}$$

Dimana:

  • $\mathbf{x}$ = Vector dari decision variables (alokasi sumber daya)
  • $\mathbf{w}$ = Vector dari objective weights
  • $\mathbf{A}$ = Constraint matrix
  • $\mathbf{b}$ = Right-hand side constraint vector

Implementation:

def optimize_resource_allocation(objective_weights, costs, budget, constraints_matrix, constraints_rhs):
    from scipy.optimize import linprog
    
    # Minimize negative objective (equivalent to maximizing)
    objective = [-w for w in objective_weights]
    
    # Solve linear programming problem
    result = linprog(
        c=objective,
        A_ub=constraints_matrix,
        b_ub=constraints_rhs,
        bounds=[(0, None)] * len(objective_weights),
        method='highs'
    )
    
    return result

6.3 Multi-Objective Optimization

Pareto Optimal Solution:

Untuk multiple objectives, kita mencari Pareto optimal solution:

$$\min_{\mathbf{x}} \quad [f_1(\mathbf{x}), f_2(\mathbf{x}), \ldots, f_k(\mathbf{x})]^T$$

Weighted Sum Method:

Menggunakan weighted sum untuk mengkombinasikan multiple objectives:

$$F(\mathbf{x}) = \sum_{i=1}^{k} w_i \cdot f_i(\mathbf{x})$$

Dimana:

  • $f_i(\mathbf{x})$ = Objective function ke-$i$
  • $w_i$ = Weight untuk objective ke-$i$
  • $k$ = Jumlah objectives

Implementation:

def multi_objective_optimization(objectives, weights, constraints):
    """
    Multi-objective optimization using weighted sum method
    """
    from scipy.optimize import minimize
    
    def objective_function(x):
        return sum(w * obj(x) for w, obj in zip(weights, objectives))
    
    result = minimize(
        objective_function,
        x0=[0.5] * len(constraints),
        constraints=constraints,
        method='SLSQP'
    )
    
    return result

6.4 Resource Allocation Optimization

Capability Development Objective:

$$\max_{\mathbf{x}} \quad \sum_{i=1}^{n} w_i \cdot \text{MPC}_i(\mathbf{x})$$

Budget Constraint:

$$\sum_{i=1}^{n} c_i \cdot x_i \leq B$$

Resource Efficiency Constraint:

$$\frac{\text{MPC}_{\text{total}}(\mathbf{x})}{\sum_{i=1}^{n} c_i \cdot x_i} \geq \epsilon$$

Dimana:

  • $\text{MPC}_i(\mathbf{x})$ = MPC score untuk capability $i$
  • $c_i$ = Cost untuk development capability $i$
  • $B$ = Total budget
  • $\epsilon$ = Minimum efficiency threshold

7.3 Value at Risk (VaR)

Mathematical Definition:

VaR mengukur potential loss pada confidence level tertentu:

$$\text{VaR}(\alpha) = F^{-1}(\alpha)$$

Parametric VaR (Normal Distribution):

$$\text{VaR}(\alpha) = \mu + \sigma \cdot \Phi^{-1}(\alpha)$$

Dimana:

  • $\alpha$ = Confidence level (e.g., 0.05 untuk 95% confidence)
  • $\mu$ = Mean dari distribution
  • $\sigma$ = Standard deviation dari distribution
  • $\Phi^{-1}(\alpha)$ = Inverse cumulative normal distribution

Implementation:

def calculate_var(data, confidence_level=0.05):
    import numpy as np
    var = np.percentile(data, confidence_level * 100)
    return var

7.4 Expected Shortfall (ES)

Mathematical Definition:

ES mengukur expected loss melebihi VaR threshold:

$$\text{ES}(\alpha) = E[X|X \leq \text{VaR}(\alpha)]$$

Empirical ES:

$$\text{ES}(\alpha) = \frac{1}{n\alpha} \sum_{i=1}^{n} x_i \cdot \mathbb{I}[x_i \leq \text{VaR}(\alpha)]$$

Dimana:

  • $X$ = Random variable (returns/losses)
  • $\mathbb{I}[\cdot]$ = Indicator function
  • $n$ = Jumlah observations

Implementation:

def calculate_expected_shortfall(data, confidence_level=0.05):
    import numpy as np
    var = calculate_var(data, confidence_level)
    es = np.mean(data[data <= var])
    return es

7.5 Risk Assessment Framework

Risk Score Calculation:

$$R = \sum_{i=1}^{k} w_i \cdot R_i$$

Risk Level Classification:

$$\text{Risk Level} = \begin{cases} \text{Low} & \text{if } R < 0.3 \\ \text{Medium} & \text{if } 0.3 \leq R < 0.7 \\ \text{High} & \text{if } R \geq 0.7 \end{cases}$$

Risk Mitigation Strategy:

$$\text{Effectiveness} = \frac{\text{Risk Reduction}}{\text{Implementation Cost}}$$

Dimana:

  • $R_i$ = Risk score untuk factor $i$
  • $w_i$ = Weight untuk risk factor $i$
  • $k$ = Jumlah risk factors

8. PERFORMANCE METRICS DAN VALIDATION

8.1 Classification Metrics

Untuk model klasifikasi VUCA levels, kita menggunakan metrics berikut:

Confusion Matrix:

$$\mathbf{C} = \begin{bmatrix} TP & FP \\ FN & TN \end{bmatrix}$$

Accuracy:

$$\text{Acc} = \frac{TP + TN}{TP + TN + FP + FN}$$

Precision:

$$\text{Prec} = \frac{TP}{TP + FP}$$

Recall:

$$\text{Rec} = \frac{TP}{TP + FN}$$

F1-Score:

$$\text{F1} = 2 \cdot \frac{\text{Prec} \cdot \text{Rec}}{\text{Prec} + \text{Rec}}$$

Dimana:

  • $TP$ = True Positives
  • $TN$ = True Negatives
  • $FP$ = False Positives
  • $FN$ = False Negatives

Implementation:

def calculate_classification_metrics(y_true, y_pred):
    from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
    
    metrics = {
        'accuracy': accuracy_score(y_true, y_pred),
        'precision': precision_score(y_true, y_pred, average='weighted'),
        'recall': recall_score(y_true, y_pred, average='weighted'),
        'f1_score': f1_score(y_true, y_pred, average='weighted')
    }
    
    return metrics

8.2 Regression Metrics

Untuk model regresi MPC score prediction, kita menggunakan metrics berikut:

Mean Squared Error (MSE):

$$\text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

Root Mean Squared Error (RMSE):

$$\text{RMSE} = \sqrt{\text{MSE}} = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$

Mean Absolute Error (MAE):

$$\text{MAE} = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$$

R² Score (Coefficient of Determination):

$$R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}$$

Dimana:

  • $y_i$ = Actual value untuk observation $i$
  • $\hat{y}_i$ = Predicted value untuk observation $i$
  • $\bar{y}$ = Mean dari actual values
  • $n$ = Jumlah observations

Implementation:

def calculate_regression_metrics(y_true, y_pred):
    from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score
    import numpy as np
    
    mse = mean_squared_error(y_true, y_pred)
    rmse = np.sqrt(mse)
    mae = mean_absolute_error(y_true, y_pred)
    r2 = r2_score(y_true, y_pred)
    
    metrics = {
        'mse': mse,
        'rmse': rmse,
        'mae': mae,
        'r2': r2
    }
    
    return metrics

8.3 Cross-Validation

Cross-validation memastikan robust evaluation dari model performance:

K-Fold Cross-Validation:

$$\text{CV}_{\text{score}} = \frac{1}{k}\sum_{i=1}^{k} \text{Score}_i$$

Standard Error of CV:

$$\text{SE}_{\text{CV}} = \sqrt{\frac{1}{k(k-1)}\sum_{i=1}^{k}(\text{Score}_i - \text{CV}_{\text{score}})^2}$$

Dimana:

  • $k$ = Jumlah folds
  • $\text{Score}_i$ = Performance score pada fold ke-$i$

Implementation:

def perform_cross_validation(model, X, y, cv=5, scoring='accuracy'):
    from sklearn.model_selection import cross_val_score
    
    cv_scores = cross_val_score(model, X, y, cv=cv, scoring=scoring)
    
    return {
        'scores': cv_scores,
        'mean': cv_scores.mean(),
        'std': cv_scores.std()
    }

8.4 Model Validation Framework

Framework validasi yang komprehensif untuk memastikan model reliability:

Validation Score:

$$\text{Validation Score} = \alpha \cdot \text{Accuracy} + \beta \cdot \text{Stability} + \gamma \cdot \text{Generalization}$$

Model Stability Index:

$$\text{Stability} = 1 - \frac{\sigma_{\text{performance}}}{\mu_{\text{performance}}}$$

Generalization Gap:

$$\text{Generalization Gap} = |\text{Train Score} - \text{Test Score}|$$

Dimana:

  • $\alpha, \beta, \gamma$ = Weights untuk different validation aspects
  • $\sigma_{\text{performance}}$ = Standard deviation dari performance scores
  • $\mu_{\text{performance}}$ = Mean dari performance scores
